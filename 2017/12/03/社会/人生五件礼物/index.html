人生的五件礼物

发表于 2017-12-03 | 分类于 社会

#马克吐温说：人生的五件礼物，你最想选哪一件？

一个人最危险的敌人就是他自己的口舌

当真理还在穿鞋的时候，谎言就能走遍半个世界

如果所有的人都是富有的，那么所有的人都是贫穷的

人生最重要的两天就是你出生的那天和你明白自己为何出生的那天

每当发现自己和大多数人站在一边，你就该停下来反思一下

保持身体健康的唯一办法，就是吃点你不想吃的，喝点你不想喝的，以及做点你不愿做的事情

旅行是消除无知和仇恨的最好方法

人生五种恩赐

1.
人生之晨，一位善良的仙女带着他的篮子走了过来，向年轻人说道：
“这些是礼物，你挑一件吧，把其余的留下，要小心，挑选时要明智。一定要明智！在这些礼物中只有一件是有价值的。”

这五件礼物分别是： 名誉、爱情、财富、快乐和死亡。年轻人急切地说道：“没必要考虑了，我选择快乐”。

他走出家门，到社会上寻求年轻人喜爱的各种快乐。然而，他所追寻的一件接一件的东西都是昙花一现、令人失望。他感觉徒劳无功和空虚，并且这其中的每一件在消失前，都要嘲弄他一番。

最后他说：“这些年的时光都被我浪费了。如果我能够再次选择的话，我一定会明智地选择。

2.
爱情中一个小时的幸福，要用一千个小时悲伤补偿
“还剩下四件礼物。再选一次吧！一定要记住，时光飞逝，四件礼物中只有一件是珍贵的。”

那人考虑了好久，最后选择了爱情。他并没有注意到仙女眼中泛起了泪花。

很多很多年后，他坐在一间空荡荡的屋子里，身旁摆着一副棺木，自言自语地说着：

“她们一个接一个地离我而去，现在躺在这里的，是我最爱的人，也是我爱的最后一个人。爱情这个奸诈的商人，每售予我一小时的幸福，我得用一千小时的悲伤来偿付，我从心底诅咒他。”

3.
名誉，鼎盛时是诽谤的靶子，衰落时成被轻视的对象
“再选一次。”仙女说。

“时间一定教你变得睿智了，还剩下三件礼物，它们中只有一个是有价值的——记住这点，小心地选择。

早已不再年轻的他，沉思了很久，选了名誉，仙女叹息着离开了。

许多年过去，仙女站在他后面，而他则在消逝的日子里孤独地坐着，思考着。仙女知道他在想什么:

“我曾在世间鼎鼎有名，人人皆夸赞我，曾经有那么一阵子，我感觉很不错。但这是多么短暂的一阵子啊！

然后随之而来的是嫉妒，接着是诽谤、是中伤、是仇恨、是迫害。再接着是嘲笑，这是结束的开端；最后是遗憾，它是名誉的葬礼。声望，鼎盛时它是诽谤的靶子，衰落时便是被轻视和同情的对象。”

4.
它们并非礼物，只是租借物

“剩下两件礼物。不要绝望。一开始的时候只有一件礼物是有价值的，它现在仍在这儿”，仙女耐心地说。

“财富，它就是力量！”

他觉得，现在，这日子终于值得过下去了。我会大把花钱、炫耀我的财富。让那些嘲笑和轻视的人，在我面前的泥土中爬行，我会用他们的嫉妒填满我的雄心。

我将拥有所有的奢侈品、所有的欢乐、所有精神上的魅力，以及人们所珍视的身体上的满足。

短短三年过去，终于有一天，那人颤抖地坐在一间破落的阁楼里，面容憔悴、脸色苍白、眼窝凹陷且衣衫褴褛。他喃喃地说：

“它们并非礼物，只是租借物。快乐、爱情、名誉、财富，它们代表着伤痛、悲伤、耻辱、贫困，这些长久现实的短暂伪装。仙女说的对，所有的藏品中只有一件是珍贵的礼物，同那件无价之宝相比，其他的东西都显得如此卑微、廉价和低劣。

那件礼物能让你身心所遭受的痛苦、耻辱和忧伤都沉浸在无梦的、永久的睡眠中。”

5.
什么才是最有价值的礼物？

那位仙女来了，再次带来了四件礼物，但就是没有“死亡”。

她说，“我把它给了一个小孩子。虽然他什么都不知道，可他相信我，要我为他做出选择，而你却没要求我为你做出决定。”

“噢，可悲的我！还剩下什么给我？”

“只剩下你甚至都不配得到的东西，时光对你的肆意欺凌。”

人生的五种馈赠，哪一个最珍贵？马克·吐温的答案是“死亡”！时光流逝，快乐、爱情、财富、名望因生命的有限才显得弥足珍贵，人因学会珍惜才能不断成长。
